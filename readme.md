# Dashboard Bike-sharing-dataset ✨

## Setup Environment - Anaconda
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard_proyek_analisis_data_akhir.py
```
